# Booking App
Complete system of Booking rooms for Coworking platform with mobile React Native with Expo, web with React.js and backend with Node.js and MoongoDB.

## Built With

* React.js
* react-router-dom
* React Native
* Expo
* Express
* MoongoDB
* Multer
* react-navigation
* axios
* styled-component
